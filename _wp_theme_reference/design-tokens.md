# TGA Theme Design Tokens

## Colors

### Primary Colors
- **Primary Dark Blue**: `#0a1844` - Main text color, primary brand color
- **Primary Dark Blue Alt**: `#151c34` - Alternative dark blue for headings
- **Teal/Accent**: `#56d4b4` - Accent color for buttons and highlights

### Neutral Colors
- **Light Gray**: `#e7e7e7` - Backgrounds, borders
- **Medium Gray**: `#7c7c7c` - Secondary text
- **Dark Gray**: `#333` - Text color
- **Very Light Gray**: `#f3f3f3` - Light backgrounds
- **White**: `#fff` - Backgrounds, text on dark

### Other Colors
- **Text Primary**: `#0a1844`
- **Text Secondary**: `#404040`, `#666`
- **Link Blue**: `#0000EE`
- **Border Gray**: `#ccc`, `#aaa`, `#bbb`
- **Background Light**: `#eee`, `#f1f1f1`, `#e6e6e6`
- **Background Yellow**: `#fff9c0`

## Typography

### Font Families
- **Primary Font**: `'Droid Sans', sans-serif`
  - Regular: 400
  - Bold: 700 (Droid Sans Bold)
  
- **Display Font**: `'Zuume', sans-serif`
  - Medium Italic: 500
  - Bold Italic: bold
  
- **Monospace**: `monospace, monospace` (for code)
- **Icons**: `"Font Awesome 5 Free"`

### Font Usage
- Body text: Droid Sans
- Headings: Zuume (display font)
- Code/Pre: Monospace
- Icons: Font Awesome

### Line Height
- Default paragraph: `28px`

## UI Behaviors (from custom.js)

### Scroll Effects
- Header hides on scroll down, shows on scroll up
- Banner image parallax effect on scroll
- Diagonal stripe parallax on default template pages

### Interactive Elements
- Preloader animation
- Mobile menu slide-in from right
- Testimonials slider (responsive: 2 columns desktop, 1 column mobile)
- Dynamic team image heights (square aspect ratio)

### Responsive Breakpoints
- Mobile: < 550px
- Tablet: 550px - 768px
- Desktop: > 768px
